# Анализ конкурентов DiplomaGuard

## 1. Список конкурентных решений

| Конкурент | Блокчейн | Плюсы | Минусы |
|-----------|----------|-------|--------|
| **Blockcerts** (MIT) | Bitcoin / Ethereum | Открытый стандарт, реальные внедрения | Сложная настройка, плохой UX |
| **Diploma** (NEAR) | NEAR Protocol | Низкая комиссия, быстро | Привязан к NEAR, мало документации |
| **EBSI** (ЕС) | Hyperledger Besu | Господдержка, DID | Закрытый, только для госорганов |

## 2. Ключевые функции конкурентов

- Выпуск неизменяемого сертификата
- Публичная верификация по ID
- Интеграция с существующими системами (API)
- Поддержка DID (децентрализованных идентификаторов)

## 3. Отличия DiplomaGuard от конкурентов

| Аспект | Blockcerts | Diploma (NEAR) | EBSI | **DiplomaGuard** |
|--------|------------|----------------|------|------------------|
| Блокчейн | Bitcoin/Ethereum | NEAR | Hyperledger | **Ethereum Sepolia** |
| Сложность внедрения | Высокая | Средняя | Очень высокая | **Низкая** |
| Целевая аудитория | Университеты | Разработчики | Государство | **Малые вузы, курсы** |
| Поддержка MetaMask | ✅ | ❌ | ❌ | **✅** |
| Открытый код | ✅ | ✅ | ❌ | **✅ (MIT)** |
| Верификация без регистрации | ✅ | ✅ | ✅ | **✅** |

## 4. Вывод (позиционирование)

DiplomaGuard — это **простая и доступная альтернатива** для учебных заведений, которые хотят внедрить блокчейн-дипломы без сложной инфраструктуры и больших затрат.
