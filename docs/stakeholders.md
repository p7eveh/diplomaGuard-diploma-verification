# Заинтересованные лица (стейкхолдеры)

## Таблица стейкхолдеров и их потребностей

| Стейкхолдер | Роль | Потребность / Ожидание |
|--------------|------|------------------------|
| **Университет / Учебное заведение** | Эмитент дипломов | Недорогой и простой способ выпускать защищённые от подделки дипломы |
| **Студент / Выпускник** | Владелец диплома | Иметь подтверждаемый документ, делиться ссылкой с работодателем |
| **Работодатель / HR** | Верификатор | Мгновенно проверить подлинность диплома без звонков и запросов |
| **Администратор системы** | Техподдержка | Управлять смарт-контрактом, добавлять новые вузы |

## Пользовательские истории (пример)

- *"Как университет, я хочу выпускать диплом в один клик, чтобы не тратить время на бумаги."*
- *"Как студент, я хочу, чтобы мой диплом нельзя было подделать, а работодатель мог проверить его за 10 секунд."*
- *"Как HR, я хочу ввести ID диплома и сразу увидеть результат, без регистрации."*
