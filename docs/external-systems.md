# Внешние системы и обмен данными

## 1. Список внешних систем

- **MetaMask** — кошелек для подписи транзакций
- **Ethereum Sepolia** — тестовая сеть для смарт-контракта
- **Браузер пользователя** — для проверки диплома без кошелька

## 2. Форматы обмена данными

### 2.1. Выпуск диплома

Данные, которые отправляются в смарт-контракт:

```json
{
  "recipient": "0x742d35Cc6634C0532925a3b844Bc9e7595f0b367",
  "diplomaId": "DIP-2026-001",
  "metadataURI": "https://myuniversity.com/diploma.json",
  "issueDate": 1704067200
}

### 2.2. Верификация диплома

Запрос от HR:

GET https://diplomaguard.com/verify/DIP-2026-001

Ответ от системы:

```json
{
  "diplomaId": "DIP-2026-001",
  "isValid": true,
  "owner": "0x742d35Cc6634C0532925a3b844Bc9e7595f0b367",
  "fullName": "Иван Петров",
  "degree": "Бакалавр",
  "university": "Технический университет"
}

### 2.3. Если диплом не найден

```json
{
  "diplomaId": "DIP-2026-999",
  "isValid": false,
  "error": "Диплом не найден"
}

## 3. Ограничения
Сеть: Sepolia (тестовая)

Стоимость выпуска: < 0.50 USD

Время подтверждения: 15-30 секунд

MetaMask нужен только для выпуска диплома

Проверка работает без кошелька и регистрации
