# MVP (Minimum Viable Product) DiplomaGuard

## Что такое MVP для DiplomaGuard

**Одна фраза, продающая продукт:**

> *"DiplomaGuard — это простейший способ выпускать и проверять дипломы на блокчейне за 10 секунд без регистрации."*

## Состав MVP (первый релиз)

### Обязательно (должно работать):

| Функция | Описание |
|---------|----------|
| ✅ Выпуск диплома | Администратор вуза подключает MetaMask, заполняет форму, диплом записывается в смарт-контракт |
| ✅ Публичная проверка | Любой пользователь вводит ID диплома на сайте и видит: действителен / не действителен |
| ✅ Уникальный ID | Каждый диплом имеет свой ID (например, DIP-2026-001) |
| ✅ Только авторизованные вузы | Смарт-контракт проверяет, что вызывающий адрес имеет право выпускать |
| ✅ Простой интерфейс | Две страницы: "Выпуск" (с MetaMask) и "Проверка" (без кошелька) |

### Не входит в MVP (будет позже):

| Функция | Почему не в MVP |
|---------|-----------------|
| ❌ Массовый выпуск (CSV) | Можно выпускать по одному в MVP |
| ❌ Личный кабинет студента | Не обязателен для проверки |
| ❌ Мобильное приложение | Достаточно веб-сайта |
| ❌ IPFS для метаданных | Можно использовать обычный JSON |
| ❌ Основная сеть Ethereum | Sepolia бесплатна для тестов |
| ❌ API для внешних систем | Только веб-интерфейс в MVP |

## Временные рамки MVP

| Этап | Длительность |
|------|--------------|
| Смарт-контракт (Solidity) | 3 дня |
| Тестирование контракта (Hardhat) | 2 дня |
| Фронтенд (HTML + ethers.js) | 4 дня |
| Деплой на Sepolia | 1 день |
| Интеграция и отладка | 2 дня |
| **Итого** | **12 дней (~2.5 недели)** |

## Успех MVP измеряется по:

- 5 успешных выпусков дипломов (тестовых)
- 10 успешных проверок (разными пользователями)
- Время проверки < 15 секунд
- Стоимость выпуска < $0.50
