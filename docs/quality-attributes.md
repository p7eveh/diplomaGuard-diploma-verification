# Атрибуты качества и архитектурные решения DiplomaGuard

## Связь атрибутов качества с архитектурными решениями

| Атрибут качества | Требование | Архитектурное решение | Как обеспечивается |
|------------------|------------|----------------------|---------------------|
| **Доступность** | 99.9% | Фронтенд на статическом хостинге (GitHub Pages / Vercel) | CDN, отсутствие собственного бэкенда |
| **Надёжность** | Данные не теряются | Данные в блокчейне | Смарт-контракт неизменяем, ноды Ethereum |
| **Производительность** | Проверка < 15 сек | View-функция без газа | Вызов не требует транзакции, мгновенный ответ |
| **Производительность** | API < 2 сек | Статический фронтенд + прямой вызов контракта | Нет промежуточных серверов |
| **Масштабируемость** | Неограниченное число дипломов | События вместо storage | События дешевле и не лимитированы |
| **Безопасность** | Только админ выпускает | Ролевая модель в контракте | Mapping address → bool, модификатор onlyIssuer |
| **Безопасность** | Нельзя подделать диплом | Блокчейн как источник правды | Хеш транзакции, неизменяемость |
| **Расширяемость** | Добавление новых вузов | Функция addIssuer() | Только owner контракта может добавлять |
| **Юзабилити** | Проверка без регистрации | Отдельная страница без Web3 | Не требует MetaMask, работает в любом браузере |
| **Экономичность** | Выпуск < $0.50 | Хранение только хеша/ID в контракте | Метаданные вне блокчейна |

## Технические ограничения

| Ограничение | Значение | Влияние на архитектуру |
|-------------|----------|------------------------|
| Блокчейн | Только Sepolia (тест) | Нет реальных денег, бесплатный газ |
| Разработчик | 1 человек | Простая архитектура без бэкенда |
| Время на MVP | 4 недели | Только необходимый минимум |
| Хранение | Нет своей БД | Всё в блокчейне + JSON-файлы |

## Пропускная способность и нагрузка

| Параметр | Расчётное значение | Обоснование |
|----------|-------------------|-------------|
| Выпусков в секунду (макс) | 0.01 (1 в 100 сек) | Выпуск — редкая операция |
| Проверок в секунду (макс) | 10 | Проверка — частая операция |
| Трафик на проверку | ~500 bytes | JSON ответ |
| Месячный трафик | ~150 MB | При 10 000 проверок в день |

## Безопасность

| Угроза | Защита |
|--------|--------|
| Выпуск диплома неавторизованным | Модификатор `onlyIssuer` в смарт-контракте |
| Подделка метаданных | Метаданные подписаны? (опционально) / Хранятся на доверенном URI |
| Атака на фронтенд (XSS) | Статический HTML, нет ввода пользователя в опасные места |
| Потеря доступа к админскому кошельку | Только owner может добавлять новых админов |
