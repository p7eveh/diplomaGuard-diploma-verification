# diplomaGuard-diploma-verification

## Идея проекта
Децентрализованная платформа для выпуска и проверки образовательных сертификатов/дипломов с использованием смарт-контрактов (Solidity, Ethereum). Подлинность документа подтверждается за 10 секунд без звонков в университет.

## Границы проекта

### Что входит в систему (Scope):
- Выпуск сертификата (администратор/вуз)
- Публичная проверка сертификата по уникальному ID
- Просмотр истории выданных сертификатов
- Интеграция с MetaMask для подписи транзакций

### Что НЕ входит (Out of scope):
- Децентрализованное хранилище файлов (IPFS — опционально)
- Мобильное приложение
- Система платного выпуска сертификатов
- Восстановление утерянного доступа к кошельку

## Цели проекта (измеримые)
1. Время проверки сертификата — менее 15 секунд
2. Стоимость выпуска одного сертификата — не более $0.5 в gas-комиссиях (Sepolia testnet)
3. Возможность верификации без регистрации и входа
4. Гарантия неизменяемости данных: ни один выпущенный сертификат нельзя подделать или удалить

## Задачи (что будет реализовано)
- Написать и задеплоить смарт-контракт CertIssuer.sol
- Создать простой веб-интерфейс (HTML/JS + ethers.js)
- Реализовать выпуск сертификата (админская роль)
- Реализовать публичную проверку по хешу сертификата
- Настроить тестовую сеть (Sepolia / Ganache локально)

## Драйверы и мотивации
- Проблема: поддельные дипломы и долгая верификация
- Решение: блокчейн даёт прозрачность и неизменяемость
- Мотивация для пользователей: бесплатно и быстро проверить кандидата
- Мотивация для вузов: повышение доверия к выпускникам

## Технологический стек (предварительно)
- **Блокчейн:** Ethereum (Solidity 0.8.x)
- **Сеть тестовая:** Sepolia / Hardhat локально
- **Frontend:** HTML, JS, ethers.js
- **Инструменты:** VS Code, Git, Hardhat или Foundry
